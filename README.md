
This repository provides a set of audio samples from the **Blizzard 2023 segmented corpus**, including:

- **Reference recordings**
- **ZipVoice outputs**
- **Fine-tuned ZipVoice outputs**



## Directory Structure
Structure:
```
collected_samples/
├── AD_train/
│   ├── reference/
│   ├── zipVoice/
│   └── zipVoice_finetuned/
└── NEB_train/
    ├── reference/
    ├── zipVoice/
    └── zipVoice_finetuned/
```
